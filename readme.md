# TechStore — Flask E-Commerce პლატფორმა

სრულფასოვანი ონლაინ მაღაზიის ვებ-აპლიკაცია, აგებული **Flask**-ისა და **SQLAlchemy**-ის საფუძველზე. აპლიკაცია მოიცავს მომხმარებლის რეგისტრაცია/ავტორიზაციას OTP ვერიფიკაციით, Google-ით შესვლას, კალათისა და შეკვეთების მართვას, Stripe-ით გადახდებს, Cloudinary-ზე ფოტოების ატვირთვას და ადმინისტრაციულ პანელს.

---

## 🧩 ძირითადი ფუნქციონალი

### ავტორიზაცია და უსაფრთხოება
- რეგისტრაცია და შესვლა ელ-ფოსტით და პაროლით
- 4-ნიშნა OTP კოდით ელ-ფოსტის ვერიფიკაცია (ვადა — 15 წუთი)
- ვერიფიცირებული არ არის → წვდომა შეზღუდულია
- პაროლების ჰეშირება (`Werkzeug Security`)
- პაროლის აღდგენა OTP კოდით
- ანგარიშის წაშლა დამატებითი OTP დადასტურებით
- დაცული გვერდები `Flask-Login`-ის საშუალებით

### Google ავტორიზაცია (`Flask-Dance`)
- რეგისტრაცია Google ანგარიშით
- შესვლა Google ანგარიშით
- არსებული ანგარიშის დაკავშირება Google-თან

### პროდუქტების კატალოგი
- მთავარი გვერდი პაგინაციით (9 პროდუქტი გვერდზე)
- ძიება სახელით
- ფილტრაცია კატეგორიების მიხედვით (მობილური ტექნიკა, აქსესუარები, კომპიუტერის ნაწილები, პერიფერიები, მონიტორები)
- სორტირება ფასის/მარაგის მიხედვით (ზრდადობით/კლებადობით)
- პროდუქტის დეტალური გვერდი + მსგავსი პროდუქტების ბლოკი

### კალათა (AJAX / Fetch API)
- დამატება, რაოდენობის ცვლილება, წაშლა, სრული გასუფთავება — გვერდის გადატვირთვის გარეშე

### გადახდები — Stripe
- დინამიური `Checkout Session`-ის გენერაცია
- წარმატებული გადახდის შემდეგ `webhook`-ის საშუალებით:
  - მარაგი ავტომატურად მცირდება
  - შეძენილი ნივთები იშლება კალათიდან

### პროფილი
- პირადი ინფორმაცია, რეგისტრაციის თარიღი
- პაროლის შეცვლა, გამოსვლა, ანგარიშის წაშლა (OTP-ით)

### ადმინის პანელი
- წვდომა შეზღუდულია `ADMIN_MAIL`-ში მითითებული მეილით (`admin_required` დეკორატორი)
- პროდუქტების დამატება/მართვა
- ფოტოს ატვირთვა პირდაპირ მოწყობილობიდან **Cloudinary**-ის მეშვეობით (URL ავტომატურად ინახება ბაზაში)
- `Flask-Admin` პანელი — Users / Items / CartItem მოდელების სამართავად
